주요 기능

요리를 더 즐겁게 만드는 기능들

📝

간편한 레시피 저장

URL을 붙여넣기만 하면 자동으로 레시피를 가져옵니다. 직접 입력도 간편해요.

📂

카테고리 정리

한식, 양식, 디저트 등 원하는 대로 카테고리를 만들어 레시피를 정리하세요.

🔍

빠른 검색

재료나 요리 이름으로 검색하면 원하는 레시피를 바로 찾을 수 있어요.

📱

오프라인 사용

인터넷 없이도 저장된 레시피를 확인할 수 있어 어디서든 요리 가능해요.

🛒

쇼핑 리스트

레시피의 재료를 쇼핑 리스트에 추가하고 장보기를 편하게 하세요.

⏱️

요리 타이머

레시피에 맞춘 타이머로 조리 시간을 놓치지 않고 완벽한 요리를 만드세요.
